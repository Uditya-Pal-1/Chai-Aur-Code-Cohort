## Repository Overview
This repository tracks Week 00 activities for the Chai Aur Code Cohort. It includes setup files, notes, and assignments for learning Git basics.

- **Local Setup**: Clone via `git clone https://github.com/Uditya-Pal-1/Chai-Aur-Code-Cohort.git`
- **Branch**: Use `main` for stable code; create feature branches for changes.

## Key Git Commands
- Initialize: `git init`
- Add files: `git add .` (stages all) or `git add filename`
- Commit: `git commit -m "Descriptive message"`
- Push: `git push origin main`
- Pull updates: `git pull origin main`
- Status check: `git status`
- Branching: `git checkout -b new-feature` [web:2]

## Workflow Steps
1. Clone the repo.
2. Create/edit files in Week-00 folder.
3. Stage: `git add Week-00/`
4. Commit: `git commit -m "Add Week 00 notes"`
5. Push: `git push origin main`

## Contributing
- Fork the repo if external.
- Use pull requests for changes.
- Follow Chai Code guidelines: Commit often with clear messages. [web:2]

## Troubleshooting
- Conflicts? Resolve manually, then `git add` and commit.
- Remote issues: `git remote -v` to verify origin URL.
